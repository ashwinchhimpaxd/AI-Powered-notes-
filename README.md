# 🧠 AI Powered Notes Website

An intelligent note-taking app built with **React**, **Appwrite**, and integrated with **AI capabilities** for content suggestions, summaries, and rewrites. Secure authentication and real-time interactions make writing smarter and easier than ever.

---

## 🚀 Features

- ✍️ Rich Text Editing with Tiptap
- 🤖 AI Chatbox for content generation and editing
- 🧠 Summarize, rewrite, or expand selected text using AI
- 🔐 Secure authentication with Appwrite
- 💾 Cloud-based notes storage (user-specific)
- 🌈 Responsive and intuitive UI
- 📁 Image upload and content embedding support
